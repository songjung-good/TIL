<npx와 npm의 차이>
- npm 
  - package Manager(관리)
  - 전역에 패키지 업데이트
- npx
  - package Runner(실행)
  - 패키지의 최신버전 파일을 불러와 설치하여 실행시키고 실행된 이후에 해당 패키지를 제거하는 방식
  - 로컬에서 필요한 패키지를 업데이트
